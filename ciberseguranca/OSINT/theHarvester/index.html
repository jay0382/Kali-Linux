theHarvester

Introdução

theHarvester é uma ferramenta de coleta de informações públicas (OSINT - Open Source Intelligence) que auxilia profissionais de segurança, pentesters e pesquisadores a encontrar dados expostos sobre domínios e empresas. Ela realiza buscas em diversas fontes abertas para coletar subdomínios, e-mails, hosts, dados do Shodan, GitHub e muito mais.

Principais Parâmetros

  • -d : Define o domínio alvo. Obrigatório)
  • -l : Limite de resultados. Padrão: 500.
  • -S : Começa na posição X nos resultados. Padrão: 0.
  • -b : Define as fontes de busca (Google, Bing, DuckDuckGo, Shodan, etc.).
  • -p : Usa proxies definidos em proxies.yaml.
  • -s : Usa o Shodan para dados adicionais dos hosts encontrados.
  • --screenshot : Captura prints das páginas encontradas.
  • -v : Verifica hosts virtuais.
  • -e : Define um servidor DNS específico.
  • -t : Verifica possibilidade de takeover de subdomínio.
  • -r : Resolve os subdomínios encontrados via DNS.
  • -n : Faz lookup DNS.
  • -c : Realiza brute-force de DNS (descoberta de subdomínios).
  • -f : Salva os resultados em arquivos XML e JSON.

Fontes de Busca Suportadas

O theHarvester pode consultar várias fontes, incluindo: 

google, bing, duckduckgo, crtsh, shodan, github-code, virustotal, censys, yahoo, entre outros

Exemplos de Uso

Busca básica

theHarvester -d exemplo.com -b google,bing -l 100

Coleta 100 resultados no Google e Bing para o domínio "exemplo.com".

Busca com brute-force de subdomínios e DNS resolve

theHarvester -d exemplo.com -b duckduckgo -c -r -l 200

Busca no DuckDuckGo, faz brute-force e resolve subdomínios encontrados.

Busca completa com proxy e salvando resultados

theHarvester -d exemplo.com -b google,bing,crtsh -p -f resultado

Usa proxies, pesquisa em múltiplas fontes e salva os resultados em arquivos resultado.xml e resultado.json.

Aviso de Responsabilidade

O uso do theHarvester deve ser feito com responsabilidade e apenas em domínios e sistemas autorizados. Coletar dados de terceiros sem permissão pode ser considerado ilegal em diversas jurisdições. Esta página tem fins exclusivamente educacionais.
