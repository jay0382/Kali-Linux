Guia Completo do Nmap

O que é o Nmap?

O Nmap (Network Mapper) é uma ferramenta de código aberto usada para varredura de redes e auditoria de segurança. Ele permite descobrir hosts ativos, serviços, sistemas operacionais e possíveis vulnerabilidades em uma rede.

Instalando o Nmap

No Kali Linux, o Nmap já vem instalado. Mas caso precise reinstalar, use o comando:

sudo apt install nmap

Comandos Básicos do Nmap

Abaixo estão alguns comandos essenciais para escaneamento de redes:

  • nmap 192.168.1.1
    Escaneia um único IP 
  • nmap 192.168.1.1/24
    Escaneia toda a sub-rede 
  • nmap -p 80 192.168.1.1
    Verifica se a porta 80 está aberta 
  • nmap -sV 192.168.1.1
    Descobre serviços e versões rodando no IP 
  • nmap -O 192.168.1.1
    Tenta identificar o sistema operacional

Comandos Avançados do Nmap

Alguns comandos avançados para auditoria de segurança:

  • nmap -A 192.168.1.1
    Escaneamento completo (OS, versão, scripts) 
  • nmap -Pn 192.168.1.1
    Ignora ping e assume que o host está ativo 
  • nmap --script=vuln 192.168.1.1
    Busca vulnerabilidades conhecidas 
  • nmap -sS 192.168.1.1
    Escaneamento stealth (SYN Scan) 
  • nmap -sU 192.168.1.1
    Escaneamento de portas UDP 
  • nmap -A -T4 192.168.10.1/24
    Escaneamento completo com informações detalhahas como, serviços ativos e suas versões, Sistema Operacional, mapeia a rede, executa um traceroute, ajuda na auditoria de segurança.

Explicação do Comando:

  • sudo → Necessário para acessar informações mais detalhadas de portas e SO.
  • nmap → Executa o scan.
  • -A → Ativa detecção de SO, versão dos serviços, scan de scripts NSE e traceroute.
  • -T4 → Define a velocidade do scan como "agressiva", tornando-o mais rápido sem exagerar na detecção de IDS/IPS.
  • 192.168.10.1/24 → Escaneia a faixa inteira de 192.168.10.1 até 192.168.10.255.
  • Veja o resultado do scan

Conclusão

O Nmap é uma ferramenta essencial para profissionais de segurança e redes. Seu uso responsável pode ajudar a identificar falhas antes que atacantes explorem vulnerabilidades.
