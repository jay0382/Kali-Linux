Estilizando Elementos com CSS

No CSS, você pode estilizar elementos usando classes ou IDs:

Classes (class):

  • São referenciadas no CSS com um ponto (.).
  • Usadas para aplicar estilos a múltiplos elementos com a mesma classe.
.category { background-color: lightgray; }

IDs (id):

  • São referenciados no CSS com uma cerquilha (#).
  • Usados para aplicar estilos a elementos únicos.
#caminhos-diretorios { font-weight: bold; }

Como criar gradientes personalizados

Sintaxe básica do gradiente linear:

background: linear-gradient(direction, color1, color2, ...);
  • direction: Define a direção do gradiente (ex.: to right, to bottom, 45deg, etc.).
  • color1, color2, ...: As cores do gradiente.

Estilos Explicados

1. font-family: Define a fonte usada no texto.

font-family: Arial, sans-serif;

2. line-height: Controla o espaçamento entre linhas.

line-height: 1.6;

3. font-size: Define o tamanho da fonte do texto.

font-size: 16px;

4. color: Define a cor do texto.

color: #ffffff; /* Branco */

5. background-color: Define a cor de fundo do elemento.

background-color: #000000; /* Preto */

6. margin: Adiciona espaço ao redor do elemento.

margin: 20px auto; /* 20px em cima/baixo, centralizado horizontalmente */

7. padding: Adiciona espaço interno dentro do elemento.

padding: 10px;

8. border: Adiciona uma borda ao redor do elemento.

border: 1px solid #00ffff; /* 1px de largura, sólido, cor ciano */

9. border-radius: Arredonda os cantos do elemento.

border-radius: 8px;

10. box-shadow: Adiciona uma sombra ao redor do elemento.

box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);

11. text-align: Alinha o texto dentro do elemento.

text-align: center;

12. opacity: Define a transparência do elemento.

opacity: 0.8;

13. display: Controla o tipo de exibição do elemento.

display: flex;

14. flex-direction: Define a direção dos elementos dentro de um flex container.

flex-direction: column;

15. position: Define a posição do elemento na página.

position: absolute;

16. top / left / right / bottom: Define o posicionamento de um elemento posicionado.

top: 20px; left: 10px;

17. cursor: Define o tipo de cursor ao passar sobre um elemento.

cursor: pointer;

18. z-index: Define a ordem de sobreposição do elemento.

z-index: 10;

19. overflow: Controla o comportamento do conteúdo que ultrapassa os limites do elemento.

overflow: hidden;

20. visibility: Controla a visibilidade do elemento.

visibility: hidden;
