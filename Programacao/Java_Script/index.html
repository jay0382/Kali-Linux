Guia Completo de JavaScript

JavaScript é uma linguagem de programação poderosa usada para criar interatividade em páginas web.

Para mais informações sobre JavaScript, acesse a página do w3schools

1. Introdução ao JavaScript

JavaScript é uma linguagem interpretada usada principalmente no desenvolvimento web para tornar as páginas interativas.

// Exemplo básico de JavaScript
console.log("Olá, JavaScript!");

2. Variáveis: var, let e const

JavaScript tem três formas principais de declarar variáveis.

  • var: Escopo global ou de função.
  • let: Escopo de bloco (recomendado).
  • const: Variável imutável.
var nome = "João";  // Pode ser redeclarado
let idade = 25;  // Pode ser modificado, mas não redeclarado no mesmo escopo
const cidade = "São Paulo";  // Não pode ser alterado

3. Estruturas de Controle: if, else, while

Podemos controlar o fluxo do código com `if`, `else` e loops.

let numero = 10;
if (numero > 5) {
    console.log("O número é maior que 5");
} else {
    console.log("O número é menor ou igual a 5");
}

let contador = 0;
while (contador < 5) {
    console.log("Contagem: " + contador);
    contador++;
}

4. Funções e Retorno de Valores

Funções permitem a reutilização de código.

function somar(a, b) {
    return a + b;
}
console.log(somar(5, 3));

5. Manipulação do DOM

Podemos modificar o HTML com JavaScript.

document.getElementById("meuElemento").innerText = "Novo texto";

6. Eventos e Interatividade

Eventos permitem interações do usuário.

document.getElementById("botao").addEventListener("click", function() {
    alert("Botão clicado!");
});

7. Promises e Async/Await

Promises são usadas para operações assíncronas.

async function buscarDados() {
    let resposta = await fetch("https://api.exemplo.com/dados");
    let dados = await resposta.json();
    console.log(dados);
}
