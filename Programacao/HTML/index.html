Introdução ao HTML

HTML (HyperText Markup Language) é a base da construção de páginas web. Aqui você encontrará exemplos detalhados e explicações para cada elemento.

O que são Tags em HTML?

Em HTML, tags são elementos usados para estruturar e organizar o conteúdo de uma página da web. Elas definem desde títulos, parágrafos e imagens até tabelas, formulários e interatividade com scripts. Cada tag tem uma função específica e pode conter atributos que modificam seu comportamento.

Principais Tags do HTML

Abaixo está uma lista das principais tags usadas no HTML, incluindo as que foram mencionadas.

  • <html></html> → Define o documento HTML.
  • <head></head> → Contém metadados, título e links para CSS e scripts.
  • <title></title> → Define o título da página (aparece na aba do navegador).
  • <body></body> → Contém todo o conteúdo visível da página.
  • <h1> a <h6> → São cabeçalhos (títulos), onde <h1> é o maior e <h6> o menor.
  • <p></p> → Define um parágrafo de texto.
  • <a href="URL"></a> → Cria links para outras páginas.
  • <img src="imagem.jpg" alt="descrição"> → Insere imagens.
  • <ul></ul> → Lista não ordenada (com marcadores).
  • <ol></ol> → Lista ordenada (com números).
  • <li></li> → Itens de listas.
  • <div></div> → Container genérico para agrupar elementos.
  • <span></span> → Elemento inline usado para estilizar partes do texto.
  • <form></form> → Define um formulário.
  • <input> → Campo de entrada em formulários.
  • <button></button> → Botão clicável.
  • <script></script> → Adiciona código JavaScript na página.
  • <link> → Conecta a página a um arquivo externo, como CSS.
  • <meta> → Define metadados, como charset e descrição da página.

Tags de Tabela em HTML

As tabelas são usadas para exibir dados de forma organizada. Aqui estão as principais tags:

  • <table></table> → Define uma tabela.
  • <thead></thead> → Cabeçalho da tabela (contém as colunas de título).
  • <tbody></tbody> → Corpo da tabela (contém os dados principais).
  • <tr></tr> → Define uma linha na tabela.
  • <th></th> → Define um título de coluna (cabeçalho de uma tabela).
  • <td></td> → Define uma célula (conteúdo dentro da tabela).

Tags de Estrutura

  • <html>: Elemento raiz de qualquer documento HTML.
  • <head>: Contém informações sobre o documento.
  • <body>: Contém o conteúdo visível da página.
  • <main>: Representa o conteúdo principal da página.
  • <section>: Divide o conteúdo em seções temáticas.

Exemplo básico de estrutura HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Minha Página</title>
</head>
<body>
    <main>
        <section>
            <h1>Bem-vindo!</h1>
            <p>Este é o conteúdo principal.</p>
        </section>
    </main>
</body>
</html>
Minha Página

Bem-vindo!

Este é o conteúdo principal.

Links e Navegação

Use a tag <a> para criar links:

<<a href="https://www.google.com" target="_blank">Visite o Google</a>>
        
Visite o Google

Atributos importantes:

  • href: Define o destino do link.
  • target: Controle de abertura (_self, _blank).

Guia Completo: Formulários, Mídia e Modais

Formulários

Os formulários permitem a entrada de dados do usuário e suportam diversos tipos de campos.

1. Formulário com Texto e Senha

<!-- Este é um formulário básico com campos de texto e senha -->
<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <br>
    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha">
</form>

2. Formulário com Seleção e Botões

<!-- Este é um formulário com uma lista de opções e botões de ação -->
<form>
    <label for="opcoes">Escolha uma opção:</label>
    <select id="opcoes" name="opcoes">
        <option value="1">Opção 1</option>
        <option value="2">Opção 2</option>
    </select>
    <br>
    <input type="submit" value="Enviar">
    <input type="reset" value="Limpar">
</form>

Reprodutores de Mídia

HTML permite a inclusão de áudio e vídeo diretamente nas páginas.

1. Reprodutor de Áudio

<!-- Reproduz arquivos de áudio com controles básicos -->
<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Seu navegador não suporta áudio.
</audio>

2. Reprodutor de Vídeo

<!-- Reproduz arquivos de vídeo com controles e largura ajustável -->
<video controls width="400">
    <source src="video.mp4" type="video/mp4">
    Seu navegador não suporta vídeo.
</video>

Modais

Os modais são janelas pop-up usadas para exibir informações ou solicitar ações do usuário.

1. Estrutura Básica de Modal

<!-- Estrutura de um modal com sobreposição e botão de fechar -->
<div class="modal-overlay" id="modal-overlay"></div>
<div class="modal" id="modal">
    <p>Este é um modal!</p>
    <button onclick="fecharModal()">Fechar</button>
</div>
<button onclick="abrirModal()">Abrir Modal</button>

<script>
    function abrirModal() {
        document.getElementById('modal').style.display = 'block';
        document.getElementById('modal-overlay').style.display = 'block';
    }
    function fecharModal() {
        document.getElementById('modal').style.display = 'none';
        document.getElementById('modal-overlay').style.display = 'none';
    }
</script>

Uso de < e > no HTML

No HTML, os símbolos < (menor que) e > (maior que) são usados para delimitar as tags HTML. Essas tags são essenciais para estruturar documentos HTML, como:

<p>Este é um parágrafo.</p>

No entanto, quando você precisa exibir esses símbolos diretamente na página (em vez de serem interpretados como tags), deve usar suas entidades HTML correspondentes:

  • &lt;: Representa o símbolo de "menor que" (<).
  • &gt;: Representa o símbolo de "maior que" (>).

Exemplo de uso

Se você quiser exibir o código HTML <p>Texto</p> como texto na página, pode usar:

<p>Texto</p>

Ao renderizar no navegador, aparecerá assim:

<p>Texto</p>

Quando usar < e >

  • Exibindo código HTML no navegador: Para mostrar exemplos de código sem que o navegador os interprete como tags.
  • Inserção segura de conteúdo: Em alguns casos, para evitar problemas de segurança, como ataques XSS, é necessário usar as entidades para que tags HTML sejam tratadas como texto.

Essas entidades são muito úteis ao criar tutoriais, documentação técnica ou qualquer página que precise exibir código-fonte HTML.
