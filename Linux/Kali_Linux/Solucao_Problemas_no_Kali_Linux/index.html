Ativando Teclas de Mídia no Linux com xbindkeys

Guia completo para configurar volume, play/pause, stop, e mais.

1. Verificar se as teclas estão sendo detectadas

Abra o terminal e digite:

xev | grep keycode

Aparecerá uma janelinha branca. Pressione as teclas desejadas (ex: volume +). Se o sistema mostrar algo como:

keycode 123 (XF86AudioRaiseVolume)

Significa que a tecla é reconhecida pelo sistema.

2. Instalar os pacotes necessários

No terminal, digite:

sudo apt update
sudo apt install xbindkeys amixer playerctl -y

3. Criar o arquivo de configuração do xbindkeys

xbindkeys --defaults > ~/.xbindkeysrc

Edite com:

nano ~/.xbindkeysrc

No final do arquivo, adicione:

# Play/Pause
"playerctl play-pause"
  XF86AudioPlay

# Stop
"playerctl stop"
  XF86AudioStop

# Próxima faixa
"playerctl next"
  XF86AudioNext

# Faixa anterior
"playerctl previous"
  XF86AudioPrev

# Aumentar volume
"amixer -q sset Master 5%+"
  XF86AudioRaiseVolume

# Diminuir volume
"amixer -q sset Master 5%-"
  XF86AudioLowerVolume

# Mudo
"amixer -q sset Master toggle"
  XF86AudioMute

4. Iniciar o xbindkeys

No terminal, execute:

xbindkeys

Para reiniciar após edição:

killall xbindkeys && xbindkeys

5. Fazer iniciar automaticamente com o sistema

Se você usa XFCE:

  • Vá em Configurações > Sessão e Inicialização
  • Acesse a aba "Início automático de aplicativos"
  • Clique em Adicionar e preencha:

Ou use o terminal e execute o comando:

 xfce4-session-settings
Nome: xbindkeys
Descrição: Atalhos de volume e mídia
Comando: xbindkeys
