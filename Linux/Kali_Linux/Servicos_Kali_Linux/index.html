Comandos para Gerenciar Serviços no Kali Linux

Ação Comando
Ver statussudo systemctl status <serviço>
Iniciar serviçosudo systemctl start <serviço>
Parar serviçosudo systemctl stop <serviço>
Reiniciar serviçosudo systemctl restart <serviço>
Ativar no bootsudo systemctl enable <serviço>
Desativar do bootsudo systemctl disable <serviço>
Bloquear serviçosudo systemctl mask <serviço>
Desbloquear serviçosudo systemctl unmask <serviço>
Listar serviços rodandosudo systemctl list-units --type=service --state=running
Listar todos os serviços disponíveis no sistemasudo systemctl list-unit --type=service
Ver quais serviços estão escutando portas específicassudo ss -tulnp
