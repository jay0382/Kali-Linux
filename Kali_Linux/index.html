Caminhos Para Diretórios Importantes no Kali Linux

  • cd /usr/share/backgrounds – Imagens de fundo da área de trabalho.
  • cd /var/lib/mysql – Banco de dados MySQL.
  • cd /etc – Arquivos de configuração do sistema.
  • cd /var/log – Logs do sistema (auditoria e eventos).
  • cd /usr/share/wordlists – Wordlists (listas de senhas).
  • cd /usr/bin – Executáveis dos comandos instalados.
  • cd /home/usuário – Pasta do usuário comum.
  • cd /root – Pasta do root (superusuário).
  • cd /tmp – Arquivos temporários, apagados ao reiniciar.
  • cd /opt – Softwares instalados manualmente.
  • cd /mnt – Montagem manual de discos e pendrives.
  • cd /media – Montagem automática de mídias removíveis.
  • cd /boot – Inicialização do sistema (GRUB, kernel).
  • cd /lib/modules/$(uname -r)/ – Módulos do kernel ativo.
  • cd /var/www/html – Diretório de páginas web (Apache2).
  • cd /etc/network – Configurações de rede e interfaces.
  • cd /etc/apt – Gerenciador de pacotes apt.
  • cd /usr/share/icons – Ícones e temas do sistema.

Manipulação de Diretórios e Arquivos

  • ls – Lista arquivos e diretórios.
  • ls -l – Lista com detalhes (permissões, proprietário, grupos).
  • cd [diretório] – Muda para o diretório especificado.
  • mkdir [nome] – Cria um novo diretório.
  • rm [arquivo] – Remove um arquivo.
  • rmdir [diretório] – Remove um diretório vazio.
  • rm -r [diretório] – Remove diretório e todo seu conteúdo.
  • mv [arquivo] [novo_nome] – Renomeia arquivo ou diretório.
  • mv [arquivo] [subdiretório]/ – Move para subdiretório atual.
  • mv [arquivo] /caminho/destino/ – Move para outro diretório.
  • mv /origem/arquivo /destino/ – Move de fora para outro diretório.
  • mv *.txt /destino/ – Move arquivos com extensão específica.
  • mv * /destino/ – Move todos os arquivos do diretório atual.
  • mv /origem/* /destino/ – Move tudo de um diretório específico.
  • cat [arquivo] – Exibe conteúdo de um arquivo.
  • echo "mensagem" > arquivo.txt – Cria um arquivo .txt com conteúdo.
  • echo "mensagem" | sudo tee arquivo.txt > /dev/null && sudo chmod 666 arquivo.txt – Cria arquivo com permissões ajustadas.
  • echo "mensagem" | sudo tee -a arquivo.txt > /dev/null && sudo chmod 666 arquivo.txt – Adiciona linha ao arquivo com permissões.
  • open [arquivo] – Abre arquivo em modo gráfico.
